# /images — EET-GS Lab Media Assets

Place all site images in this folder.

## Recommended files

| File | Usage |
|------|-------|
| `favicon.png` | Browser tab icon (32×32 px) |
| `Nahid.png` | Photo of Lab Director Dr. Nahid (used in members page) |
| `placeholder.png` | Generic avatar for members without a photo |
| `og-image.jpg` | Open Graph preview image for social sharing (1200×630 px) |

## Member photos

Name each member photo consistently, e.g.:
- `member-firstname-lastname.jpg`

Then update the `src` attribute in `members/index.html` accordingly.

## Notes
- Prefer `.webp` or optimised `.jpg/.png` for performance
- Aim for member portraits at 200×200 px minimum, square crop
